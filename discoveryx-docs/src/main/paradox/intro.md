# 介绍

*Fusion DiscoveryX 成长于作者对 Akka 生态的学习，API上参考了Nacos的某些设计。*

服务注册与发现管理系统，基于Akka生态开发。

## 技术

| 功能       | 使用技术              |
| ---------- | --------------------- |
| 开放API    | Akka gRPC             |
| 集群序例化 | Protobuf              |
| 配置持久化 | Akka Persistence      |
| 容错与扩展 | Akka Cluster Sharding |
| REST       | Akka HTTP             |
